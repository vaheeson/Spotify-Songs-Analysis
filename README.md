# 🎵 Spotify Songs Data Analysis Project

## 📌 Overview
This project explores Spotify song data to identify trends in music popularity and understand how different audio features influence a track’s success. The analysis includes data cleaning, exploratory data analysis (EDA), and visual storytelling using Python.

---

## 🎯 Objectives
- Analyze key audio features such as danceability, energy, tempo, and loudness  
- Identify patterns that influence song popularity  
- Perform data cleaning and preprocessing  
- Visualize relationships between variables for better insights  
- Summarize findings for data-driven decision making  

---

## 📂 Dataset
The dataset contains Spotify track information including:
- Track name and artist  
- Popularity score  
- Audio features (danceability, energy, valence, tempo, etc.)  
- Duration and other metadata  

---

## 🛠️ Tools & Technologies
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 🔄 Project Workflow
1. Data Loading  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Feature Analysis  
5. Data Visualization  
6. Insights & Conclusions  

---

## 📊 Key Insights
- Certain audio features like **energy and danceability** show correlation with popularity  
- Most songs fall within specific tempo and duration ranges  
- Popular tracks often share similar audio feature patterns  
- Clear relationships exist between mood-related features and listener preference
